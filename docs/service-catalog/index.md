# Service Catalog

The Service Catalog provides a functioning menu of implementation support services for public-sector customers. Each service includes purpose, triggers, inputs, outputs, and turnaround expectations.

## How to Use

1. Select the service that matches your current domain challenge.
2. Prepare required inputs listed on the service page.
3. Submit an intake request using the service request template.
4. Track status and outputs in your project workspace.
