# Service 02: Data Readiness Assessment

## Purpose
Assess whether available data is sufficient, governed, and fit for intended analytics use in Domain III.

## When to Use
- Data quality or access concerns may block delivery.
- Multiple source systems require harmonization.
- Governance and stewardship controls are unclear.

## Required Inputs
- Data source inventory
- Access constraints and stewardship contacts
- Current analytics problem statement

## Outputs
- Data readiness findings report
- Quality and governance risk log
- Remediation plan and Domain IV readiness recommendation

## IAF Task Coverage
- **Primary tasks:** 3.1, 3.2, 3.3, 3.4, 3.5, 3.6, 3.7, 3.8
- **Supporting domains:** Domain II validation and Domain IV readiness

## Legal Guardrail
All service execution must comply with applicable law and policy, including the Foundations for Evidence-Based Policymaking Act of 2018. See `docs/public-sector/legal-guardrails`.

## Standard Turnaround
- 10-15 business days, based on data access scope.
