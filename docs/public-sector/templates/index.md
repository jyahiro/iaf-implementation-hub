# Digital Templates

These templates are digitally enabling artifacts designed for direct use in repository workflows. Copy a template into your project workspace and complete it as part of domain progression.

## Usage Standard

- Keep each artifact in Markdown.
- Track changes through version control.
- Link completed artifacts to the relevant domain page.
- Include owner, review date, and approval status in every artifact.
