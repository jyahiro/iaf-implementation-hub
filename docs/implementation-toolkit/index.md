# Implementation Toolkit

The Implementation Toolkit combines digital templates and support services into a single operating workflow for public-sector delivery teams.

## How It Works

1. Select the appropriate template and draft required artifacts.
2. Request the matching service when facilitation, validation, or readiness review is needed.
3. Update artifacts with service outputs and decision records.
4. Use domain checklists to confirm readiness before advancing.
